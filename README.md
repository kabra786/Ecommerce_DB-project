# 🛒 E-Commerce Database Management System

A complete E-Commerce Database Management System developed using:

- Python
- Streamlit
- Oracle Database (Oracle XE)
- SQL
- Database Management System Concepts

---

## 📌 Project Overview

This project provides an Admin Dashboard for managing an E-Commerce platform.

The system allows administrators to:

- Manage Products
- Manage Customers
- Manage Inventory
- Manage Orders
- View Analytics and Reports
- Monitor Business Performance

---

## 🚀 Features

### 📦 Product Management
- Add Products
- Update Products
- Delete Products
- Search Products
- Category Management

### 👥 Customer Management
- View Customers
- Search Customers
- Customer Purchase History
- Customer Statistics

### 📊 Inventory Management
- Track Stock Levels
- Update Inventory
- Low Stock Alerts
- Inventory Reports

### 🧾 Order Management
- View Orders
- Track Order Status
- Order Analytics

### 📈 Analytics Dashboard
- Total Revenue
- Total Orders
- Best Selling Products
- Revenue by Category
- Low Stock Analysis

## ▶️ Installation

### Clone Repository

```bash
git clone https://github.com/kabra786/Ecommerce_DB-project.git
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run app.py
```
